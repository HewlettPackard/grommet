## Stack
Stacks components on top of the first child component.

## Usage

```javascript
import { Stack } from 'grommet';
    <Stack/>
```

## Properties

**anchor**

Where to anchor children from. If not specified, children fill the
      first child's area.

```
center
left
right
top
bottom
top-left
bottom-left
top-right
bottom-right
```
